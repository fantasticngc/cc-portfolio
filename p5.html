Project #5 Optical Illusion -- Moon in Front of The Window

Click Here to open source code

Description

Name: Shuyu Xu


Work Description:

Every civilization has its own traditional pattern to show the cultural backgrounds. This project mainly focuses on creating an optical illusion by using the traditional patterns to explore the boundaries of visual illusions and the combination of patterns and illusions.

Design Process

First, I drew 3 traditional patterns in Photoshop.

The first illustration uses the lightest blendmode to make it look like a neon circle in the middle. However, the neon parts of the circle is still white. This illusion will appear when the background lines are messy and the neon part of the lines' edges can form a circle. People can see this illusion due to the complements of the primitives. And the traditional pattern comes from the Chinese windows' pattern.


































The second illusion is that people can see the blurry dots in the cross of the white lines. And the pattern is a kind of Budanism pattern.





























































The third illusion is the glitch pattern of thunder cloud. When mouse clicked, the pattern will show through the multiply blendmode function.

































Finally, I chose the first illusion. I drew a window with the traditional Chinese window pattern. Then used an ellipse to symbolize the moon. When clicking the mouse, the scene will become from night to day. And the moon will look like a shadow.

Reflection

In order to decline the lines of code, simplyfing the logics of code is the most crucial part. In this week's tutorial, for and while loops will assist me to transfer a long list of regular coding into several conditions, which makes it easier to draw much more complecated graphics.

Using loop statement can also help me with organize the logics of my code. This method makes my code clear than before and be easily to read except for the object-oriented programming and defining functions. Besides, another advantage of using loops is that when I change one variable's value, it is more convenient to change less variables.

When I drawing the pattern, I divided the pattern into two small patterns in order to array them conveniently. And I also use conditional statement to remove some values which I don't need in the loops. The combination of different statements will create endless possibilities.
