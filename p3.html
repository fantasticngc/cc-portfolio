Project #3 Experimental Clock Stellaris

Click Here to open source code

Description

Name: Shuyu Xu


Work Description:

Conventional clocks mark seconds, minutes, and hours according to a 24 hour cycle. However, the time in everybody's mind varies from each other. Means to define the time can also be different. In this project, I define the time in my mind and transfer it into an illustration. Because time is not only an object but also an emotional subject in my mind.

Design Process

First, I drew 3 illustrations in Photoshop. The images and psedudocodes are below.





























This is my first illustration. The circles represent stellaris in the universe and the triangles mean the stars. As ancient people use celestrial bodies to define the time whatever the civilization is, the movement and glow help ancient people to create clocks. Hence, I drew an abstruct illustration about stellaris in the sky to define the time.

Pseudocode:

let s = second();
let m = minute();
let h = hour();

draw some ellipses and change the colors, locations, sizes according to varibales s, m, h.
if h<7 or h>18, day = false
else day = true

change the background color when day boolean is changed.





























Then, I drew a halo and aura and a celetrial. In the center of the aura the number of time will fall down as time goes by.

Pseudocode:

let s = second();
let m = minute();
let h = hour();

draw an ellipse and nofill, only use stroke.
draw another ellipse and make it rotate around the halo -- translate the coordinate (0,0)to the rotate center
when time number is changed, the canvas will create the new number and fall down to the bottom.





























The third image is an illustration with three candles. When time goes by, the candle will be shorter and shorter and the background will change as the real time is night.

Pseudocode:

let s = second();
let m = minute();
let h = hour();
let heightS = 100;
let heightM = 100;
let heightH = 100;

the height of the candle will be shorter as time gose by.
heightS = heightS - s
heightM = heightM - m
heightH = heightH = h

Finally, I chose the first idea and combined some ideas of other two.

When the time is night, the background will become dark blue.





























When the time is day, the background will become red.































Some ellipses will rotate and change the sizes. The yellow stars will shine in the background.

Reflection

In my perspective, I wish to describe the time as an abstruct illustration due to the time is abstruct. So it reminds me of the paintings of Kandinsky. Actually, when we do not need accurate time to calculate things, time only symbols the passing of memories, the changes of colors, and the flows of emotions. Hence, I used the stellaris illustration to represent time. Because ancient people use them to define time though it is not such accurate as nowadays.

As my code is much more complex and longer than the previous 2 projects, I learned how to comment my code as neant as possible and how to use multiple functions to clarify different logic parts. And using boolean to define a event makes the judge much more easier. If-else statement also can help me to organize some logics to change varibles' values.
