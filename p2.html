Project #2 Face Generator

Press mouse button to generate faces

Click Here to open source code

Description

Name: Shuyu Xu


Work Description:

Parameterized design is creating some variables and generating various results by changing the parameters. This work will use random function and mousepressed function to generate various values and create different faces.

Design Process

First, I drew a maximum face in Photoshop. The image is below.





























Then, I drew a minimum face. I changed some the positions of some points in these curves.





























After finishing two faces, I drew the maximum face in p5.js. Click Here to open source code

I created some variables to substitute the original position values and created mousePressed function to change the value of positions and colors by random function.

If I want to constrain some primitives in a specific area, I should use add, substruct, multiply and divide to link two related variables.

Reflection

At first, I feel so confused how to define a variable of my work. Then, I realized the variables are what I can control in my coding. And define the variables smartly will reduce a lot of effort to access to the same result and simplify the code.

When finishing this work, it is interesting that everything can be revised in coding. Arranging the logic and math relationship between various variables could be the most essential part.
